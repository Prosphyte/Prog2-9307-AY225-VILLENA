Programming 2 Machine Problem Set

Java and JavaScript CSV Dataset Processing

Students will create programs that process the provided CSV dataset located in this repository. Each student must complete three (3) assigned machine problems. Programs must be implemented using both Java and JavaScript.

General Instructions

  1. Pull this repository from Git.
  2. Locate the dataset file provided in the repository.
  3. Your program must FIRST ask the user to enter the dataset file path before executing any processing.
  4. All machine problems must be implemented in:
    • Java
    • JavaScript
  5. Programs must correctly read and parse CSV data.
  6. Each program must contain clear comments explaining:
    • Variables
    • Functions
    • Processing logic
    • Dataset handling
  7. Comments are required to help the instructor evaluate the code.
  8. Output must be clearly formatted and readable.
  9. Each submission must include:
    • Source code
    • Screenshot of program output
    • Short explanation of the program logic (3–5 sentences)

Java Program Flow

  1. Start Program
  2. Prompt user for dataset file path
  3. Read CSV using BufferedReader and FileReader
  4. Store records using ArrayList or objects
  5. Process dataset depending on machine problem
  6. Display formatted output
  7. Handle errors (missing file, invalid data)
  8. End Program

JavaScript Program Flow

  1. Start Program
  2. Prompt user for dataset location
  3. Read file using Node.js File System module
  4. Parse CSV data into arrays or objects
  5. Process dataset based on machine problem
  6. Display formatted results
  7. Handle possible errors
  8. End Program

Machine Problems

MP01 – Load dataset and display total number of records.
MP02 – Display the first 10 rows of the dataset.
MP03 – Search for a keyword in the dataset.
MP04 – Count valid rows excluding empty rows.
MP05 – Extract and display a selected column.
MP06 – Display unique values in a column.
MP07 – Sort records alphabetically by a column.
MP08 – Filter records using a keyword.
MP09 – Display dataset statistics.
MP10 – Detect duplicate records.
MP11 – Frequency count for column values.
MP12 – Display dataset in formatted table output.
MP13 – Detect rows with missing values.
MP14 – Count keyword occurrences.
MP15 – Export first 50 rows to a new CSV file.
MP16 – Randomly display 10 rows from dataset.
MP17 – Find the longest text entry in dataset.
MP18 – Remove rows with empty fields.
MP19 – Generate dataset summary report.
MP20 – Convert CSV dataset into JSON format.
